# Безопасность

Не публикуйте пароль OBS WebSocket, содержимое приватных сцен и личные пути без необходимости.

Для отчёта используйте безопасный диагностический отчёт внутри приложения и создайте issue без секретов. Если публикация деталей может навредить пользователю, сначала свяжитесь с владельцем репозитория через профиль GitHub.

Приложение принимает телеметрию только от локальных виджетов, запрещает произвольную навигацию окна и не сохраняет пароль OBS WebSocket.
