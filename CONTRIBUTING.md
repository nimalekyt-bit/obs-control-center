# Участие в разработке

Спасибо за интерес к OBS Control Center. Проект находится в Early Access, поэтому особенно полезны воспроизводимые отчёты с реальными конфигурациями Windows и OBS.

## Перед созданием Issue

1. Проверьте, что похожая задача ещё не открыта.
2. Не публикуйте пароль OBS, токены, приватные сцены и личные пути.
3. Для ошибки используйте встроенный безопасный диагностический отчёт и просмотрите его перед отправкой.
4. Укажите версию приложения, Windows и OBS Studio.

## Pull Request

1. Создайте отдельную ветку от `main`.
2. Ограничьте изменение одной задачей.
3. Добавьте тест для исправленной ошибки или новой гарантии.
4. Выполните `npm test` и `npm audit --omit=dev`.
5. Не добавляйте моковые сцены, телеметрию или личные данные в production UI.

Отправляя вклад, вы соглашаетесь распространять его по лицензии MIT проекта.
